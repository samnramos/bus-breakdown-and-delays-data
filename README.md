# bus-breakdown-and-delays-data


## To Obtain the data sets

- Create a new folder
- Open https://data.cityofnewyork.us/Transportation/Bus-Breakdown-and-Delays/ez4e-fazm/about_data for bus breakdowns and delays data set
- Export the data into a csv file import it into the folder you just created.

## What you need

- `Bus_Breakdown_and_Delays_20260720.csv` - dataset with air quality and health impact data.
- `bus_breakdown.ipynb` - Jupyter notebook for analyzing the dataset.
- Make sure python, seaborn, pandas, and plotly in installed so the data runs smoothly.
